# landing-page
New landing page for portfolio

## Install Dependencies
```sh
$ npm install
```

## Run dev mode
First edit webpack.config.js file and edit mode to development, run
```sh
$ npm run start
```

## Build and deploy
```sh
$ npm run build
$ npm run deploy
```
